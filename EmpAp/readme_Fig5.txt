This zip file includes MATLAB code to reproduce Figure 5 inGoncalves, Herrera, Kilian and Pesavento (2021), forthcoming in the Journal of Econometrics. 
It uses the plug-in estimator for the block recursive structural model discussed in that paper.

The main file is Fig5.m. The file includes instructions on the definition of the various inputs and the output. The 
remaining files are called up when executing plugin.m.
